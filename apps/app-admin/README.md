# Workix Admin Frontend

Administrative dashboard for Workix platform management.

## Overview

Angular-based admin interface for managing users, pipelines, integrations, and platform configuration.

## Port

- **Port**: `7300`
- **URL**: `http://localhost:7300`

## Technologies

- Angular 20+
- PrimeNG
- Zoneless (Signals)
- Shared UI components (`libs/shared/frontend/ui`)

## Features

- User management
- Pipeline administration
- System configuration
- Analytics & monitoring
- RBAC management
