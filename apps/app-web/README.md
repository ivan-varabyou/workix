# Workix Web Frontend

Client-facing web application for Workix platform.

## Overview

Angular-based web interface for creating, running, and monitoring AI-driven workflows.

## Port

- **Port**: `7301`
- **URL**: `http://localhost:7301`

## Technologies

- Angular 20+
- PrimeNG
- Zoneless (Signals)
- Shared UI components (`libs/shared/frontend/ui`)

## Features

- Visual pipeline editor
- Workflow execution
- Dashboard & monitoring
- User profile management
- Integration management
