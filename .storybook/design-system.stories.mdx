# Workix Design System

Complete design system for modern web applications built with Angular 20, signals, and accessibility-first approach.

## Color Palette

### Primary Colors
- **Primary Blue**: `#007bff` - Main brand color, CTAs, links
- **Secondary Gray**: `#6c757d` - Secondary actions, text
- **Success Green**: `#28a745` - Success messages, confirmations
- **Danger Red**: `#dc3545` - Errors, destructive actions
- **Warning Yellow**: `#ffc107` - Warnings, cautions
- **Info Cyan**: `#17a2b8` - Information, tooltips

### Neutral Colors
- **White**: `#ffffff` - Backgrounds, surfaces
- **Light Gray**: `#f9f9f9` - Subtle backgrounds
- **Gray**: `#e9ecef` - Borders, dividers
- **Dark Gray**: `#495057` - Secondary text
- **Black**: `#000000` - Primary text

## Typography

### Font Stack

## Components Library

### Button Variants
- **Primary**: Main actions
- **Secondary**: Alternative actions
- **Success**: Positive actions
- **Danger**: Destructive actions
- **Ghost**: Minimal/transparent buttons
- **Disabled**: Inactive state

### Sizes
- **Small** (sm): 32px height, 8px padding
- **Medium** (md): 40px height, 12px padding
- **Large** (lg): 48px height, 16px padding

## Animations & Transitions

### Standard Durations
- **Fast**: 100ms - Quick interactions
- **Normal**: 200ms - Standard transitions
- **Slow**: 300ms - Important transitions
- **Slower**: 500ms - Emphasis effects

### Easing Functions
```
cubic-bezier(0.4, 0, 0.2, 1) - Material ease
cubic-bezier(0.25, 0.46, 0.45, 0.94) - Custom ease
```

## Icons

Icon size scale:
- **16px** - Inline with text
- **24px** - Default UI icons
- **32px** - Large icons
- **48px** - Extra large, hero icons

## Accessibility

### Color Contrast
- **Normal text**: Minimum 4.5:1 contrast ratio
- **Large text**: Minimum 3:1 contrast ratio
- **UI components**: Minimum 3:1 contrast ratio

### Focus States
- All interactive elements must have visible focus indicators
- Focus outline: 2px solid with high contrast color
- Outline offset: 2px

### Keyboard Navigation
- Tab order follows visual flow
- All controls accessible via keyboard
- Enter/Space for activation

## Responsive Breakpoints

```
Mobile:   0px - 639px
Tablet:   640px - 1023px
Desktop:  1024px - 1919px
XL:       1920px+
```

## Dark Mode

Automatically applied based on:
1. System preference (`prefers-color-scheme`)
2. User setting (persistent in localStorage)
3. Time-based switching

### Dark Mode Colors
- Background: `#1a1a1a`
- Surface: `#2d2d2d`
- Text: `#ffffff`
- Muted: `#999999`
